# Store Assets pour ATOMIC FLIX

Ce dossier contient tous les assets nécessaires pour publier ATOMIC FLIX sur Amazon Appstore.

## Assets requis

### 1. Icônes
- **icon.png** - 512x512px (icône principale)
- **adaptive-icon.png** - 512x512px (Android adaptive icon)
- **small-icon.png** - 114x114px (petite icône)

### 2. Screenshots
Minimum 3 screenshots requis :
- **screenshot-1.png** - Page d'accueil avec recherche
- **screenshot-2.png** - Page de détails anime
- **screenshot-3.png** - Lecteur vidéo
- **screenshot-4.png** - Lecteur manga (optionnel)

### 3. Graphics
- **feature-graphic.png** - 1024x500px (bannière principale)
- **promo-graphic.png** - 180x120px (promotion)

## Dimensions exactes requises

### Screenshots mobile :
- **Portrait** : 1080x1920px ou 1080x2400px
- **Format** : PNG ou JPEG
- **Qualité** : Haute résolution

### Screenshots tablet (optionnel) :
- **Portrait** : 1200x1920px
- **Format** : PNG ou JPEG

## Guidelines pour les screenshots

1. **Screenshot 1** - Page d'accueil
   - Montrer la recherche d'anime
   - Interface principale visible
   - Logo ATOMIC FLIX en évidence

2. **Screenshot 2** - Détails anime
   - Page d'un anime populaire
   - Informations détaillées
   - Boutons de lecture visibles

3. **Screenshot 3** - Lecteur vidéo
   - Interface de lecture
   - Contrôles vidéo
   - Qualité HD mise en avant

4. **Screenshot 4** - Lecteur manga
   - Interface de lecture manga
   - Navigation entre pages
   - Qualité des images

## Conseils pour de bons screenshots

- Utiliser des animes populaires (Naruto, One Piece, etc.)
- Éviter le contenu inapproprié
- Montrer l'interface en action
- Utiliser des images de haute qualité
- Pas de texte superposé
- Respecter les dimensions exactes

## Outils recommandés

- **Design** : Figma, Canva, Photoshop
- **Screenshots** : Appareil Android ou émulateur
- **Compression** : TinyPNG pour optimiser les tailles

## Checklist avant soumission

- [ ] Toutes les icônes aux bonnes dimensions
- [ ] Minimum 3 screenshots de qualité
- [ ] Feature graphic attrayante
- [ ] Pas de contenu inapproprié
- [ ] Texte lisible sur tous les assets
- [ ] Cohérence avec l'identité ATOMIC FLIX
